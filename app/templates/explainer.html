Bet Grading System Explained

Grade Components

Composite Score Breakdown

  • Expected Value (EV) Score: 55%
  • Timing Score: 15%
  • EV Trend Score: 15%
  • Bayesian Confidence: 15%

The grading system prioritizes Expected Value (EV) as the primary indicator of bet quality, while considering timing, trend analysis, and Bayesian confidence to provide a comprehensive assessment.

Bayesian Confidence

Bayesian confidence is calculated by considering multiple factors:

  • Initial EV as prior probability
  • EV change over time:
    • Positive changes: 0.5x multiplier (small boost)
    • Negative changes: 1.0x multiplier (larger penalty)
  • Time-based adjustments:
    • Early improvements (>20hrs): 5% boost
    • Late-stage changes (<3hrs):
      • Positive/Stable: 10% boost
      • Negative: 50% penalty
  • Long-term stability bonus:
    • 5% boost for bets stable over 12+ hours
    • Applies only if EV change < 10%

Bell Curve Grading

Grades are assigned using a bell curve distribution based on the last 10,000 bets:

  • A: Top 2.5% (z-score ≥ 2.0)
  • B: Next 13.5% (1.0 ≤ z-score < 2.0)
  • C: Middle 68% (-1.0 < z-score < 1.0)
  • D: Next 13.5% (-2.0 < z-score ≤ -1.0)
  • F: Bottom 2.5% (z-score ≤ -2.0)

EV Override Rules

  • EV ≥ 20%: Cap at 'C' (suspicious high EV)
  • EV ≥ 10%: Minimum grade 'A'
  • EV ≥ 5%: Minimum grade 'B'

Distribution Statistics

The grading system maintains rolling statistics:

  • Uses last 10,000 bets for distribution
  • Trims top and bottom 2.5% to remove outliers
  • Updates every hour to maintain accuracy
  • Falls back to absolute grading if distribution data unavailable
